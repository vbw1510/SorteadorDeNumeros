# 🎲 Sorteador de Números

Um projeto simples e intuitivo desenvolvido para realizar sorteios de números aleatórios diretamente no navegador.

## 🚀 Sobre o projeto

O Sorteador de Números permite gerar números aleatórios de forma rápida e prática, ideal para:

- Sorteios
- Jogos
- Decisões aleatórias
- Estudos de lógica de programação

O projeto foi desenvolvido utilizando tecnologias simples mas com intuito de aprendizado e prática 

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
